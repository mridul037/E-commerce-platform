Senior Software Engineer

Dailyobjects is currently looking for a senior software engineer to join our e-commerce team in New Delhi. The right candidate will become part of our highly motivated development team and will design and develop complex web applications as well as modules for our e-commerce platform.

Your responsibilities will include building and maintaining high-performance, scalable e-commerce systems ready for the cloud. As a senior software engineer, you will develop elegant, maintainable solutions to complex problems using the principles of agile software development and you will write clean code using PHP as well as work closely with product management, interaction designers and UI engineers.

Primary Functions:

  • Perform optimization of existing and future Product implementation.
  • Troubleshoot, maintain, and configure server environment.
  • Perform integration between Magento and other applications.
  • Develop new technology through custom development
  • Collaborate with marketing department to develop new ways to enhance functionality and increase conversions.

Mandatory Skills:

  • Develop PHP web applications with Object Oriented MVC architecture.
  • Establish Code development standards.
  • Develop code optimized for optimum performance.
  • Develop web service Clients/Servers utilising SOAP/XML-RPC.
  • Develop Simple/Advanced SQL queries.
  • Develop Database Stored Procedures, Transactions/Rollbacks.
  • Optimize SQL Queries and database to improve the performance.
  • Collaborate with the HTML & Design Team to develop the Frontend.
  • Front-end development utilising JavaScript (jQuery/Prototype) and AJAX.
  • Develop completely customised upgradable websites in Magento.
  • Upgrade existing websites to newer Magento versions.
  • Extension development in Magento.
  • Participate in Design and Development of new features in the product.
  • Responsible for the deployment of the web application
  • Understanding client requirements & functional specifications.
  • Effectively communicate status, metrics and issues on a regular basis.
  • Execute assigned tasks in a timely & efficient manner.
  • Assist team members & colleagues to solve technical issues.

Necessary skills and qualifications

  • A university degree in computer science or a similar academic background. Impress us with your profound knowledge in PHP5, Magento Core Architecture, MySQL, Zend Framework & MVC, HTML, CSS and JS.
  • A structured approach and experience in object-oriented development as well as a disciplined approach to testing and quality assurance as well as experience in implementing software engineering best practices (e.g. unit testing, code reviews, design documentation).
  • A solid understanding of the full web technology stack (e.g. HTTP, cookies, headers, asset loading / caching). In addition you should have experience with web infrastructure and fault-tolerant distributed systems.
  • Must have a minimum of 4-5 years of development experience with Magento including knowledge of best practices pertaining to the fundamentals of Magento development (extending existing functionality, developing custom extensions, etc.)
  • Must have at least 3 years of PHP development.
  • Must be an expert in MySQL.
  • Must have experience with writing Javascript, CSS, jQuery, and HTML.
  • XML experience is required.
  • Must have excellent written and verbal communications skills.
  • Linux experience is required.
  • Magento Developer and Zend Certification are preferred.

Location: South Delhi

Salary Range: Industry standard

Role: Senior Software Engineer
